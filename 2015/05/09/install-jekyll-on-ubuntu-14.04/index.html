Ubuntu 14.04 安装Jekyll正确姿势

  • jekyll
  • linux
  • ubuntu

Another:Anysun
转载请注明出处，谢谢

前提

sudo apt-get install ruby ruby-dev

本能

sudo apt-get install jekyll

会产生如下错误：

/usr/lib/ruby/1.9.1/rubygems/custom_require.rb:36:
in `require’: iconv will be deprecated in the future, use
String#encode instead.

导致错误的原因可能是由于缺少相关依赖 remove掉重新安装

安装nodejs以及最新ruby

sudo apt-get install ruby ruby-dev make gcc nodejs

安装Jekyll

还是使用gem安装，虽然有些不爽：（

在不修改gem源的情况下，推荐还是使用VPN，要不你懂的

Tips:引用之前某牛，天朝的Coder是全世界最有耐心的Coder

sudo gem install jekyll –no-rdoc –no-ri

Tips:Install the Jekyll gem system wide. For speed, we are excluding the extended documentation. To include all documentation, omit the –no-rdoc –no-ri switches.

运行jekyll

jekyll -v

无错

Good Luck :)
