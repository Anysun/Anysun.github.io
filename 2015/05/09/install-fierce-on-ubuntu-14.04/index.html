从Kali Linux移植Fierce到Ubuntu 14.04

  • fierce
  • kali
  • linux
  • ubuntu

Author:Anysun
转载请注明出处，谢谢

1、安装相应的依赖库

sudo apt-get install libnet-dns*

2、移植Fierce.pl

从Kali的/usr/bin/fierce 复制到Ubuntu的 /usr/bin/fierce

sudo cp test.pl /usr/bin/fierce

从Kali /usr/bin/fierce/usr/share/fierce/hosts.txt 复制到Ubuntu的 /usr/bin/fierce/usr/share/fierce/hosts.txt

sudo mkdir /usr/share/fierce
sudo cp hosts.txt /usr/share/fiercehosts.txt

That´s All!

Good Luck :)
